# NTToolKit

A collection of extensions I find helpful in addition to Lorem/Person string generation.

## Requirements

- iOS 9.1+
- Xcode 8.0+

## Installation

#### CocoaPods
You can use [CocoaPods](http://cocoapods.org/) to install `NTToolKit` by adding it to your `Podfile`:

```ruby
pod 'NTToolKit'
```
